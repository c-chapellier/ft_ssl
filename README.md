# ft_ssl
Recode some cryptographic algorithms.
- md5
- sha256

Used functions
- open
- close
- read
- write
- lseek
- malloc
- free
- printf
- strcmp
- strlen
- mmap

md5
- -p, echo STDIN to STDOUT and append the checksum to STDOUT
- -q, quiet mode
- -r, reverse the format of the output
- -s, print the sum of the given string

sha256
- -p, echo STDIN to STDOUT and append the checksum to STDOUT
- -q, quiet mode
- -r, reverse the format of the output
- -s, print the sum of the given string
